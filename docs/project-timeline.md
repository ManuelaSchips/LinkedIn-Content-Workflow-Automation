# Project Timeline

This timeline documents the development of the LinkedIn content workflow automation project created during my time at Procycons.

The project started from a narrow brief — automate LinkedIn carousel posts based on predefined job roles — and evolved into a complete workflow system with structured requests, role-based matching, AI-generated drafts, review logic, and SharePoint integration.

## Development Summary

Within a short project window, I moved from:

- reviewing source documents
- learning n8n in project context
- designing the workflow logic

to building and presenting a functioning content workflow system with:

- SharePoint request intake
- structured job role modelling
- topic-to-role matching logic
- LLM-based carousel generation
- draft handling and review flow
- user-oriented operational process

The project demonstrates fast ramp-up in a new tool environment, structured solution design, and independent implementation of a business-relevant workflow system.
